# Product Brief: Japan Trip Planner

## Overview

**Project Name:** Japan Trip Planner
**Industry:** Travel
**Product Type:** Productivity Tools
**Platform(s):** Web

## The Idea

A Japan trip planning web app that helps first-time visitors create personalized itineraries based on their trip length, group size, and activity preferences. The app solves the problem of overwhelming choices and lack of local knowledge by providing curated recommendations with visual inspiration, interactive maps, and flexible scheduling tools.

## Target Audience

First-time Japan travelers across all demographics - from young backpackers to families with kids to luxury travelers. Anyone planning their inaugural trip to Japan who needs guidance on where to go, what to do, and how to organize their time effectively.

## Key Functions

1. Trip Preferences Input - Booking-style interface where users select travel dates, number of travelers, and activity preferences (temples, food, nightlife, shopping, etc.)
2. Personalized Itinerary Generation - System creates custom recommendations based on user inputs, displaying activities as cards that can be added, removed, and reordered via drag & drop
3. Visual Attraction Gallery - Each activity shows thumbnail images and opens detailed popups with photo galleries and descriptions of what to do at each location
4. Interactive Map View - Displays pins for all activities with recommended routes between locations, including transport options toggle (driving vs public transport) similar to Google Maps
5. Schedule/Timeline View - Simplified daily sequences showing activities in order (e.g., 'Day 1: Temple → Lunch → Shopping District → Dinner') with activity catalog sidebar for adding more options

## User Flows

### 1. Trip Preferences Input

**Happy Flow:**
Scene 1 — Homepage: User lands on clean homepage with familiar booking-style interface. Scene 2 — Form Input: User selects trip dates with date picker, chooses number of travelers from dropdown, selects activity preferences via checkboxes (temples, food tours, shopping, nightlife). Scene 3 — Submit: User clicks 'Plan My Trip' button, system shows loading screen while generating recommendations.

### 2. Personalized Itinerary Generation

**Happy Flow:**
Scene 1 — Results Page: User sees generated itinerary with activity cards organized by day. Scene 2 — Customization: User drags activities to reorder, removes unwanted activities with X button. Scene 3 — Adding Activities: User browses activity catalog sidebar, clicks to add new activities to specific days. Scene 4 — Save: System auto-saves changes, user sees confirmation.

### 3. Visual Attraction Gallery

**Happy Flow:**
Scene 1 — Activity Cards: User sees thumbnail images on each activity card in itinerary. Scene 2 — Detail Popup: User clicks activity card, popup opens showing photo gallery and detailed description. Scene 3 — Browse Gallery: User swipes through multiple images, reads about what to do at location. Scene 4 — Close: User closes popup, returns to itinerary view.

### 4. Interactive Map View

**Happy Flow:**
Scene 1 — Map Toggle: User clicks map view button from itinerary page. Scene 2 — Activity Pins: User sees map with pins for all selected activities, routes connecting locations. Scene 3 — Transport Options: User toggles between driving and public transport, sees updated route recommendations. Scene 4 — Pin Details: User clicks map pin, sees activity info popup with option to edit.

### 5. Schedule/Timeline View

**Happy Flow:**
Scene 1 — Timeline Toggle: User clicks timeline view from itinerary page. Scene 2 — Daily Sequences: User sees simplified daily breakdown (Day 1: Temple → Lunch → Shopping District). Scene 3 — Activity Catalog: User browses sidebar catalog of additional activities organized by type. Scene 4 — Add to Day: User drags activities from catalog to specific days, sees updated timeline sequence.

## Competitor Research

### TripIt

**Key Features:**
- Automatic itinerary creation by forwarding confirmation emails to plans@tripit.com
- Document management (PDFs, photos, boarding passes, digital passport QR codes)
- Schedule coordination showing where to be and when
- Travel stats tracking
- Easy sharing and calendar sync
- Offline access to itineraries
- Pro version ($49/year) with real-time alerts, alternate flight suggestions, seat tracking

**Market Position:**
- Nearly 20 million users on world's highest-rated travel planner app
- Works with any booking platform - organizes travel plans regardless of where you book
- Strong focus on business travel and trip organization

**Our Differentiation:**
- **Japan-Specific Expertise:** TripIt organizes existing bookings but doesn't suggest what to book. Our app provides curated Japan recommendations for first-time visitors.
- **Visual Discovery:** TripIt lacks visual inspiration. Our app features photo galleries and attraction images to help users discover and get excited about destinations.
- **Interactive Planning:** TripIt is post-booking organization. Our app is pre-booking planning with drag & drop customization and activity catalog browsing.
- **Cultural Context:** TripIt doesn't provide local insights. Our app will include Japan-specific cultural tips and transportation guidance.
- **Beginner-Focused:** TripIt assumes you know what to book. Our app guides first-time visitors through the entire discovery and planning process.
